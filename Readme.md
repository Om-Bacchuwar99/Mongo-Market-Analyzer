##### ‚öôÔ∏è Project Workflow \& üìÅFiles



The project is structured into two main stages:



1.Data Ingestion (*ingest\_data.py*)



* Action: Downloads over four years of daily historical data for \*\*Microsoft (MSFT)\*\* using the `yfinance` library.
* Database Use: Stores the data in a dedicated \*\*MongoDB Time Series Collection\*\*, which is optimized for high-volume, time-stamped financial data.





2\. Analysis and Visualization (*analyze\_visualize.py*)



* Action: Queries the MongoDB collection.
* Aggregation: Utilizes the efficient MongoDB Aggregation Framework to calculate the 50-Day SMA directly within the database.
* Result: Generates and saves a high-quality, interactive Candlestick Chart using Plotly to visually compare the daily price against the calculated SMA.
